## Installation

#### Clone the project

```bash
  git clone https://github.com/bahatarlaci/next-ticketing-app.git
```

#### Go to the project directory

```bash
  cd next-ticketing-app
```

#### Install TicketingApp with npm

```bash
  npm install
```

#### Run:

```bash
  npm run dev
```
